Import Instructions

Upload a CSV file with the following columns (first row must be headers):

  • unit_number (required, unique)
  • plate_number
  • make, model, year
  • status (ACTIVE, DOWN, or RETIRED)
  • current_mileage_km
  • current_engine_hours
  • chassis_number, engine_number
  • fuel_type, tank_capacity_liters
  • CR fields: mv_file_no, cr_number, or_number, or_expiry, etc.
  • notes

If a truck with the same unit_number exists, it will be updated. Otherwise a new truck is created.
